# ReactAdmin
React Admin

參考: [React-Admin Tutorial](https://marmelab.com/react-admin/Tutorial.html)

# 建立專案
參考: [Installing React-admin With Vite](https://marmelab.com/react-admin/Vite.html)
```bash
npx create-vite@latest my-admin -- --template react
```

# 執行
```bash
cd my-admin
npm install
npm run dev
```

# 瀏覽
```bash
http://localhost:5173/
```
